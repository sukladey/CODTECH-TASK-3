# CODTECH-TASK-3


## Dashboard Development

### Objective
The objective of this project is to create an interactive dashboard to visualize a dataset and generate meaningful insights.

### Tools Used
Python
Plotly Dash
Pandas
VS Code

### Dataset
A sample sales dataset containing information about:
- Region
- Product
- Category
- Sales
- Profit

### Dashboard Features
The dashboard includes the following visualizations:

1. Sales by Region (Bar Chart)
2. Profit by Category (Pie Chart)
3. Sales by Product (Bar Chart)

### How to Run the Project

1 Install dependencies

pip install -r requirements.txt

2 Run the dashboard

python dashboard.py

3 Open browser

http://127.0.0.1:8050/

### Insights

Electronics products generate the highest sales.
Laptop sales dominate in East and North regions.
Furniture category provides moderate profits.
